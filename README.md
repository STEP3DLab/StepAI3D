# StepAI3D

Статический сайт мастер-класса по ИИ и 3D с рабочим 3D-вьюером и формой заявки.

## Что сделано

- Подключены CSS и основной JavaScript в `index.html`
- Реализован 3D-вьюер для загрузки FBX-модели
- Добавлена проверка доступности ресурсов на локальном сервере
- Усовершенствована отправка формы: сначала `/api/submit`, затем fallback на `mailto:` для статического хостинга
- Добавлен легкий Node-сервер для локальной разработки и публикации

## Быстрый запуск

### Локально через Node

```bash
npm install
npm start
```

Откройте `http://localhost:3000`

### Локально как статический сайт

```bash
python3 -m http.server 8000
```

Откройте `http://localhost:8000`

## Развертывание

- Для статического хостинга (GitHub Pages, Netlify и т.п.) используется статический артефакт `_site`.
- Рабочий деплой на GitHub Pages выполняется через `.github/workflows/deploy.yml`.
- Если нужен серверный API для формы, на Node-хостинге запускается `server.js`.

## Примечания

- `api/submit.js` сохраняет логи отклоненных заявок в папку `logs`, но она игнорируется через `.gitignore`.
- Форма сначала пробует отправить заявку на `/api/submit`; при отсутствии сервера открывается `mailto:`.
- Файлы `assets/Meshy_AI_11111_1777620302_texture.stl`, `.vscode/` и `logs/` были удалены как ненужный мусор.
